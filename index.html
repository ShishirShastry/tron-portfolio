PNEUMOVIT

PneumoViT is a deep learning-based chest X-ray diagnostic tool that uses a fine-tuned Vision Transformer (ViT-B/16) built with PyTorch to classify images into Pneumonia, Lung Cancer, Tuberculosis, and Normal cases. It features a Gradio-based web interface and a standalone macOS desktop app built using Electron.js for offline access. A Convai-powered conversational AI agent is integrated to explain predictions in natural language. The system incorporates Explainable AI methods such as Grad-CAM, Integrated Gradients, and ViT attention maps for interpretability. It also generates clinical-style reports with confidence scores and maintains session history with exportable logs. Technologies used include PyTorch, Gradio, Electron.js, Convai, Captum, OpenCV, and PIL.
